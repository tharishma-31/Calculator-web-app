# Calculator Web App
A responsive calculator web application built using HTML, CSS, and JavaScript. Features arithmetic operations, percentage calculations, delete and clear functions, dark/light mode toggle, and a mobile-friendly responsive design.

## Features
- Basic arithmetic operations
- Percentage calculations
- AC and DEL functionality
- Light/Dark mode toggle
- Responsive design for mobile and desktop
- User-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
